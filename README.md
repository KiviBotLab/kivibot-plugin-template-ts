# kivibot-plugin-template-ts

`KiviBot` 插件 TS 开发模板，开箱即用，近乎零配置。

## 特征

- ✅ 预安装必要的开发依赖
- 🔧 完备的 TS 编译配置
- 🎨 简洁明了的模板代码
- 💻 开发友好的调试方式（`tsc` + `tsc-alias` + `concurrently`）
- 🧪 可选配置的测试套件（`ava`）

## 开发

使用 `degit` 克隆到本地进行开发

```shell
npx degit KiviBotLab/kivibot-plugin-template-ts kivibot-plugin-ts-demo
```
